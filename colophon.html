threshold

colophon.

this is a small private journal made public. it collects loose thoughts about mathematics, physics, and the soft objects that live between them. essays, notes, lists, fragments, and marginalia. nothing is finished. that is on purpose.

the site is set in times new roman, with an instrument serif grace note for titles. the only colours are ink, paper, baby pink for selections, and a powder blue for links. nothing else.

the drawing on the homepage is an ascii field that morphs between three states forever: a cross-stitch sampler, a fibonacci spiral, and an electron-orbital. it is rendered character by character in a single <pre> tag. there is no canvas, no webgl, no images. the page weight is mostly text.

long essays sometimes carry small ascii interludes between sections — the same morph engine, but scroll-driven rather than time-driven. they complete one full transition as you read past them.

the site is three html files, one css file, and one javascript file. no build tools, no framework, no dependencies. to add a new entry, open site.js and add an object to the ENTRIES array at the top.

if you came here from somewhere, thank you for coming. if you came here from nowhere, thank you for that too.
